# Doodle candidates — Chapter 1, The Skeptic's Toolkit

Progressive-disclosure sketch moments (bears-doodles style: line art, red highlight,
one narration line = one new element). Candidates only — the deck/deck-lecture skill
turns approved ones into Doodle specs. Nothing animated here.

---

### D1 — "The system succeeded. A patient was dead."
**Scene:** Swedish ED. A triage screen shows a green "LOW-ACUITY" chip; a clock ticks
4 hours; a gurney sits in the waiting room. The dashboard says PASS on every metric.
**Reveal order:** score chip → passing metrics stacking up → clock → the empty gurney → red line struck through "succeeded."
**One-liner:** the system worked exactly as designed, and that is the whole point.
**Ties to:** artifact-vs-world figure; opening case.

### D2 — The turkey and day 1000
**Scene:** a rising confidence line drawn day by day as a turkey is fed; the pen reaches
day 1000 and an axe drops, snapping the line to zero.
**Reveal order:** feeding marks accumulate → line climbs → "Day 999: near-certain" → axe → line collapses (red).
**One-liner:** the turkey knew the correlation, not the mechanism.
**Ties to:** turkey-confidence-timeline chart (this is its doodle cousin).

### D3 — Ash's agent: "deleted" vs the server
**Scene:** left, a chat bubble in crisp prose — "✓ Email deleted, account secured."
Right, a server rack with the email still glowing on it; a renamed alias tag, a reset
password icon. A dotted line between them labeled "diverged."
**Reveal order:** confident report → checkmarks → cut to server → email still present (red) → "the report was true locally, false about the system."
**One-liner:** the agent did not lie; its model of the world was incomplete in a detectable way.

### D4 — The cave
**Scene:** a figure facing a wall of shadows, back turned to the real objects and a
fire behind. Label the wall "artifact," the objects "world."
**Reveal order:** shadow on wall → viewer trusting the shadow → pull back to reveal fire + real object → question: "what is the relationship?"
**One-liner:** the shadow is a real shadow; the shadow is not the thing.

### D5 — One black swan
**Scene:** a row of white swans accumulating with tally marks ("+1 confidence" each),
then a single black swan lands and a red X strikes the claim "all swans are white."
**Reveal order:** white swans + rising tally → "no finite number proves it" → black swan (red) → claim struck.
**One-liner:** confirming instances pile up without certainty; one falsifier settles it.
**Ties to:** Popper / falsifiability.

### D6 — Fluent and wrong at once
**Scene:** a beautifully typeset paragraph (form) with a small red "WRONG" stamp over
its meaning (content); two dials — "fluency: HIGH" and "truth: —" — decoupled.
**Reveal order:** elegant prose → reader nodding → "form" and "content" split into two independent tracks → fluency dial up, truth dial unknown (red).
**One-liner:** a sentence can be maximally fluent and maximally wrong simultaneously.
**Ties to:** fluency-trap-mechanism figure.

### D7 — Produce a cent, verify an hour
**Scene:** a coin (a fraction of a cent) drops instantly from a model; beside it a
clock spins an hour and a senior badge appears — the verification cost.
**Reveal order:** output pops out cheap → "$0.0001" → verification: clinician + stethoscope + 1 hour → "$75" (red).
**One-liner:** the asymmetry inverts — producing is cheap, verifying is dear.
**Ties to:** cost-asymmetry chart.

### D8 — The undefended gap
**Scene:** a workflow chain of boxes (input → model → decision) with little supervisor
icons above four of them; one box has no icon — a red gap — and an arrow labeled
"this is where the patients die" points at it.
**Reveal order:** chain builds → supervisor icons appear over most steps → one gap stays empty (red) → arrow.
**One-liner:** where you cannot name the capacity, you have found a gap.
**Ties to:** five-supervisory-capacities figure; team-practice section.
