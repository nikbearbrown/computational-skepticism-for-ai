# pantry/ — Phase 1 research notes

One `<chapter-slug>_notes.md` per chapter: the sourced facts, primary citations, and cases each
chapter needs, gathered *before* the build-first rewrite. Each note also resolves the
`[verify]` / unsourced load-bearing claims the editor pass flagged in `notes/<chapter>.md`.

Discipline (AI+1 Phase 1): every claim states a real source (author, title, year, URL/DOI),
marks primary vs. secondary, and flags anything unconfirmed as `[UNVERIFIED]` rather than
guessing. Nothing is fabricated. Agreements across sources harden; conflicts are noted for the
human review queue.
